# RustZone-Mods
Official repository for the automatic download of mods for the Minecraft 'Rust Zone' server.

# RUST ZONE Launcher Prototype

A zombie-themed Minecraft mod server launcher prototype built with Electron, Node.js, HTML, and CSS.

## Visual Theme: "Classic Apocalypse"
- Dark, gritty textures.
- Blood-red accents.
- Custom title bar with a glitchy "RUST ZONE" header.

## Features
- **Custom Frame:** Draggable window title bar with minimize and close buttons.
- **Server Status:** Live status and player count indicators.
- **Simulated Launch:** Validates username and shows a launch sequence.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your system.

### Installation
1. Clone the repository or download the source files.
2. Open a terminal in the project directory.
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Application
To launch the prototype, run:
```bash
npm start
```

## Project Structure
- `main.js`: The Electron main process.
- `src/index.html`: The main UI structure.
- `src/style.css`: The zombie-themed styling.
- `src/renderer.js`: UI logic and animations.
